GitHub 동기화 설정
토큰 발급 방법:
1. 여기를 클릭하여 GitHub 토큰 생성 페이지로 이동
2. repo 권한 체크 확인 → Generate token 클릭
3. 생성된 토큰(ghp_...)을 복사하여 위 칸에 붙여넣기

※ 토큰은 이 브라우저에만 저장되며 외부로 전송되지 않습니다.
※ 토큰 없이도 읽기는 가능합니다 (수정/저장 시 필요).

Asana 연동 설정
토큰 발급 방법:
1. Asana 개발자 콘솔로 이동
2. + Create new token 클릭 → 이름 입력 (예: PK Dashboard)
3. 생성된 토큰을 복사하여 위 칸에 붙여넣기

※ 토큰은 이 브라우저에만 저장됩니다.
※ 연동 시 내게 할당된 미완료 업무를 가져옵니다.
